# Civic Eye - Report Civic issues Instantly!!!

Civic Eye is an Android app designed to help citizens report everyday civic issues such as potholes, garbage overflow, broken streetlights, and similar public problems. The app provides a clear and simple way for users to submit an issue with a photo, description, and location.Once a report is submitted, Civic Eye allows users to track its progress step by step, bringing transparency to how civic issues are handled. By combining easy reporting with automated tracking, the app aims to improve communication between citizens and authorities and encourage responsible participation in keeping cities clean and functional.

---

## Problem

Civic issues are everywhere, but reporting them is slow and unclear.  
People do not know where to report, what happens after reporting, or if the issue is ever fixed.

---

## Solution

Civic Eye provides:
- A mobile-first way to report issues with photo, description, and location
- Automated handling of reports by the authorities, hence promotes more easy access to solutions (available when connected/supported by local authorities)
- Clear tracking of report status
- Articles that promote responsible citizenship

The goal is to improve civic engagement using simple automation and clean design.

---

## Key Features

- File a report with photo, description, and auto location
- Automated status flow (submitted, ongoing, resolved)
- Track all submitted reports in one place
- Clean onboarding and login flow
- Home screen with quick actions and citizen articles
- Simple, green-themed UI focused on clarity
